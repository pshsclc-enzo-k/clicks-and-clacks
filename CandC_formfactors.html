Types of Keyboard Form Factors

All keyboards have a size, and each size (or form factor) depends on user preference. Some keyboards are large, with keys used mainly for macros or office work. Some keyboards are small, usually used when gaming for extra mouse space. In this page, we will be going over the common types of keyboard sizes, and their different use cases.


100% or Full Sized

This is 100% layout or full-sized layout. It contains all the keys you need to use a normal computer, and is the basis for all other keyboard form factors. This layout includes the alphanumerics, modifier keys, the function row, the navigation and arrow keys, and the number pad. This layout is mainly used for office work, and is the one of the most common layout for all keyboards, even membrane or rubber dome keyboards.

1800

The 1800 layout is a more compact form of the 100%. As such, it is sometimes called the 96% layout. The right shift is a bit smaller, to compensate for the arrow cluster being closer to it. Also, the navigation keys are bunched together and now rest directly on top of the number pad. This layout is one of the least common form factors for keyboards, since people usually using full-sized layouts do not need to have their keyboards smaller.

80% or TKL

The Tenkeyless or TKL layout is one of the most common layout for custom mechanical keyboards. It is a full-sized keyboard minus the number pad, hence the name "Tenkeyless". This is mainly used for a combination of gaming and productivity, and is usually favored due to the extra space it provides for the mouse when gaming. Since the number pad is usually not needed (dut to the presence of the number row), this is a favorite among enthusiasts and average keyboard users alike.

75%

The 75% layout has seen a rise in popularity due to the releases of entry level keyboards like the GMMK Pro. It is basically a compact form of the TKL layout, with some of the navigational keys being removed to accomodate desk space. You usually never see this layout for the average keyboard, and as a rule of thumb, 75% keyboards are mainly found in the custom mechanical keyboard scene. If the layout is 75%, it is usually an entry level board, or a high-grade enthusiast piece.

65%

A 65% keyboard is the preferred layout for gamers, other than the 60%. It consists of only the essential keys for games (the alphanumeric keys and modifiers), as well as the arrow keys for work use. This layout gives gamers more space in their desk for their mice, but retains the arrow keys for productivity. Most keyboards of this size also have function layers, to accomodate for the lack of navigational keys and function row.

60%

The 60% layout is, along with the 65%, arguably the best keyboard for gamers. This keyboard is a 65% layout without the arrow keys and navigation keys, consisting only of the alphanumeric keys and modifiers. This is less practical for people who aren't into video games, and, as such, most keyboards made for gamers are usually in this form factor.

40%

Finally, the 40%. This layout is rarely seen outside of a custom mechanical keyboard, and was made for diehard enthusiasts interested in using the alphabet keys only. To some, this layout is a stylistic choice, while most see it as impractical. As some enthusiasts (like myself) will tell you, you pay more for less keyboard. This is the irony in 40% keyboards. Though they contain less keys than your average keyboard, they are usually worth more since they were made with enthusiasts who are willing to pay top dollar in mind.
